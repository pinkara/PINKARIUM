Algorithme Doomsday

Calcul mental du jour de la semaine

Principe Général

L'algorithme "Doomsday" repose sur le fait que certaines dates faciles à retenir (les "Doomsdays") tombent toujours le même jour de la semaine pour une année donnée.

$$ J_{\pi}(A) = J_{\pi}(S) + a + \left\lfloor \frac{a}{4} \right\rfloor \pmod{7} $$

Note : La formule ci-dessus est la version simplifiée. La version complète (Wikibooks) décompose $a$ en multiples de 12.

L'Ancre du Siècle $J_{\pi}(S)$

Elle dépend des deux premiers chiffres de l'année (ex: 19 pour 1985).

def J_pi_century(year):
    """Ancre du siècle (Règle des 2-0-5-3)"""
    c = (year // 100) * 100
    r = c % 400
    if r == 0: return 2   # 1600, 2000 -> Mardi
    if r == 100: return 0 # 1700, 2100 -> Dimanche
    if r == 200: return 5 # 1800 -> Vendredi
    return 3              # 1900 -> Mercredi

def is_leap(year):
    return (year % 4 == 0 and year % 100 != 0) or \
           (year % 400 == 0)

def J_pi_year(year):
    """Formule Wikibooks pour l'année"""
    a = year % 100
    century_anchor = J_pi_century(year)
    
    # Décomposition par douzaines
    dozens = a // 12
    remainder = a % 12
    fours = remainder // 4
    
    return (century_anchor + dozens + remainder + fours) % 7

def get_day(day, month, year):
    # Table des pivots (Doomsdays)
    pivots = {
        1: 4 if is_leap(year) else 3,
        2: 29 if is_leap(year) else 28,
        3: 14, 4: 4, 5: 9, 6: 6,
        7: 11, 8: 8, 9: 5, 10: 10, 
        11: 7, 12: 12
    }
    
    anchor_day = J_pi_year(year) # Jour pivot de l'année
    pivot_date = pivots[month]
    
    diff = day - pivot_date
    final_day = (anchor_day + diff) % 7
    
    days = ['Dim', 'Lun', 'Mar', 'Mer', 'Jeu', 'Ven', 'Sam']
    return days[final_day]

# --- Exemple d'utilisation ---
# Quel jour était le 14 juillet 1789 ?
print(get_day(14, 7, 1789))
# Résultat attendu : 'Mar' (Mardi)
